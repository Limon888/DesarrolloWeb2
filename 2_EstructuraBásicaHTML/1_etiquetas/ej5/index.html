Etiquetas HTML

Las etiquetas HTML hace referencia al lenguaje de marcado para la elaboración de páginas web irve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

  4. <article>

    Es una clase de palabra que acompaña al sustantivo dentro de una oración.

    • Publicación en el foro.
    • Entrada en el blog.
    • Noticia.

  6. <audio>

    Agrega controles de audio, reproducción, pausa y volumen. El <source> permite especificar archivos de audio y los puedes elegir en el navegador.

  7. <button>

    Es un botón en el que se puede hacer clic.
    Dentro de un <button> se puede poner algún texto o número, (y etiquetas como <i>, <b>, <strong>, <img>, etc.) !No se puede con un botón creado con el <input>.
    Sugerencia: especificque siempre el type de un <button>, para indicar a los navegadores que tipo debotón es.
    Consejo: se puede diseñar botones facilmente usando css.

  8. <form>

    Es un contenedor donde se puede agregar diferentes tipos de entradas como: campos de texto, casilla de verificación, botnes de opción, botones de envio, etc.

    • <input type="text">

      • Muestra un campo de entrada de texto de una sola linea

    • <input type="radio">

      • Muestra un botón de redondo (con multiples opciones).

    • <input type="checkbox">

      • Muestra un botón cuadrado (para multiples opciones)

    • <input type="submit">

      • Muestra un botón de envio (para enviar en el formulario).

    • <input type="button">

      • Muestra un botón en el que se puede hacer clic

      Ejemplo form

  9. <footer>

    Define el pie de pagina para un documento o sección.
    Normalmente contienen:

    • Información de autoría
    • Información del registo
    • Información de contacto
    • Mapa del sitio
    • Volver a los enlaces superiores
    • Documentos relacionados
